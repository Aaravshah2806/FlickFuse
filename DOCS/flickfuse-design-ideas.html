Motion Design

Animation Concepts

🎞️
Film Grain Pulse
CSS noise texture that subtly pulses in opacity — 3% to 6% over 4s. Gives the entire page a living, breathing quality without performance cost.
🌊
Scroll Velocity Blur
Cards apply a CSS `blur(4px) scale(0.98)` during fast scroll, snapping sharp when stopped. Feels physical, like inertia-based motion.
✨
Spotlight Cursor
A radial gradient "flashlight" follows the cursor on dark hero sections, revealing hidden texture beneath. Pure CSS `background: radial-gradient` on `mousemove`.
⚡
Magnetic Buttons
CTA buttons slightly translate toward the cursor when nearby (within 80px). Uses JS proximity detection + GSAP `quickTo`. Creates a tactile, premium feel.
